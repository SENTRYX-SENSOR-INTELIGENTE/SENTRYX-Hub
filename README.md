# SENTRYX HUB

Central oficial de links da SENTRYX.

---

## Sobre

O SENTRYX HUB foi desenvolvido para centralizar os principais acessos da plataforma SENTRYX em uma interface moderna, responsiva e intuitiva.

A página reúne:
- Plataforma oficial
- GitHub
- Redes sociais
- Pesquisa de campo
- Integrantes da equipe
- Links institucionais

---

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- GitHub Pages

---

## Funcionalidades

- Interface responsiva
- Cards interativos
- Preview da plataforma
- Integração com redes sociais
- Design moderno
- Animações suaves
